    dmidecode

    DMI (SMBIOS) table decoder

    get serialnumber:

    1
    dmidecode -s system-serial-number

    1
    10D3G1004B10130C2A

    get system manufacturer:

    1
    dmidecode -s system-manufacturer

    1
    IGEL Technology GmbH

    get product name:

    1
    dmidecode -t system | grep 'Product Name' | cut -c 16-

    1
    M350C

    get BIOS version

    1
    dmidecode -t bios -q
